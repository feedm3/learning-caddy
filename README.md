# Learning Caddy

This project is used to learn [caddy server](https://caddyserver.com/).

## Start

To run the project on windows 64-bit simply hit
```
start.cmd
```

## Features / Setup

- run caddy on `localhost:2020`
- server files from `www` directory
- log to `logs/access.log`
